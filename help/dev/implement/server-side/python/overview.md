# Panoramica di Python SDK

Questa guida di riferimento mostra come i clienti di [!DNL Adobe Target] possono installare, inizializzare e utilizzare Python SDK.

## Prerequisiti

Sono supportate tutte le versioni di Python attualmente gestite. Vedere [Versioni di Python](https://www.python.org/downloads/).

## Introduzione

Per informazioni su come iniziare a utilizzare Python SDK, passare alla [[!DNL Target] Guida introduttiva di SDK](../sdk-guides/getting-started/getting-started.md).
